Class RepositoryModel

  • java.lang.Object
    • net.java.truelicense.core.auth.RepositoryModel
  • Direct Known Subclasses:
    BasicRepository

    @ParametersAreNullableByDefault
@Nullable
public class RepositoryModel
extends Object
    A model for storing authenticated objects (artifacts) as encoded strings. The content type and the content transfer encoding of the encoded artifact and encoded signature are defined externally.

    This class is used by repositories to store artifacts for signing and verifying.
