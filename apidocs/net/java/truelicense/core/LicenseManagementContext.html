net.java.truelicense.core

Interface LicenseManagementContext

  • All Superinterfaces:
    CachePeriodProvider, ClassLoaderProvider, Clock, CodecProvider, CompressionProvider, LicenseAuthorizationProvider, LicenseInitializationProvider, LicenseProvider, LicenseSubjectProvider, LicenseValidationProvider, PasswordPolicyProvider, RepositoryProvider
    All Known Implementing Classes:
    BasicLicenseManagementContext, BasicV2LicenseManagementContext, V1LicenseManagementContext, V2JsonLicenseManagementContext, V2XmlLicenseManagementContext

    public interface LicenseManagementContext
extends CachePeriodProvider, ClassLoaderProvider, Clock, CodecProvider, CompressionProvider, LicenseAuthorizationProvider, LicenseInitializationProvider, LicenseProvider, LicenseSubjectProvider, LicenseValidationProvider, PasswordPolicyProvider, RepositoryProvider
    A root context for the life cycle management of license keys.

    Applications have no need to implement this interface and should not do so because it may be subject to expansion in future versions.

    Author:
    Christian Schlichtherle
