      • create

        License create(License bean,
             Sink sink)
               throws LicenseManagementException
        Generates a license key from the given license bean and stores the result to the given sink.

        Calling this operation performs an initial authorization check.

        Parameters:
        bean - the license bean to process. The bean is not modified by this method.
        sink - the sink for storing the license key.
        Returns:
        A duplicate of the initialized and validated license bean which has been stored in the license key.
        Throws:
        LicenseValidationException - if validating the license bean fails, e.g. if the license has expired.
        LicenseManagementException
