Interface LicenseConsumerContext

  • All Superinterfaces:
    ContextProvider<LicenseManagementContext>, LicenseApplicationContext

    public interface LicenseConsumerContext
extends LicenseApplicationContext
    A derived context for license consumer applications. Use this context to configure a LicenseConsumerManager with the required parameters. For a demonstration of this API, please use the TrueLicense Maven Archetype to generate a sample project - even if you don't use Maven to build your software product.

    Applications have no need to implement this interface and should not do so because it may be subject to expansion in future versions.
