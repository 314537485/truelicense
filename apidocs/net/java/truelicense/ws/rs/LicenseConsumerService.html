      • LicenseConsumerService

        public LicenseConsumerService(@Context
                      javax.ws.rs.ext.Providers providers)
        Constructs a license consumer service. This constructor immediately resolves the license consumer manager by looking up a ContextResolver<LicenseConsumerManager> in the given providers. You can provide a context resolver for this class like this: 
        package ...;

 import javax.jax.rs.ext.*;
 import net.java.truelicense.core.LicenseConsumerManager;

 &#64;Provider
 public class LicenseConsumerManagerResolver
 implements ContextResolver<LicenseConsumerManager> {
     &#64;Override public LicenseConsumerManager getContext(Class<?> type) {
         return LicenseManager.get();
     }
 }

        where LicenseManager is your custom license manager enumeration class as generated by the TrueLicense Maven Archetype.

        Parameters:
        providers - the providers.
